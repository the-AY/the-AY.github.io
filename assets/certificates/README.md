# Certificates Folder

Drop your certificate images here (`.jpg`, `.png`, `.webp`, `.pdf`) and push to GitHub.

The achievements page will **automatically detect and display** them — no code changes needed!

## Naming Convention (Optional)
Use descriptive filenames for better display:
- `AWS-Cloud-Practitioner-2024.png`
- `Google-Cybersecurity-Certificate.jpg`
- `Coursera-Python-Specialization.pdf`

The filename (without extension) becomes the display title.
Hyphens (`-`) and underscores (`_`) are converted to spaces automatically.
